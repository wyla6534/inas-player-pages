INAS Player 隐私政策

生效日期：2026 年 7 月 5 日

INAS Player 是一款本地媒体库和 NAS 媒体播放工具。我们重视你的隐私。本政策说明 App 在使用本地网络、媒体刮削、字幕、歌词、iCloud 同步、购买和缓存功能时如何处理数据。

1. 基本原则

INAS Player 不包含广告 SDK，不用于跨 App 或跨网站追踪，也不会出售你的个人数据。你添加的 NAS、WebDAV、SMB、Plex、Jellyfin、Emby、IPTV 等来源仅用于连接、扫描、展示和播放你选择的媒体。

2. 本地网络

App 会请求本地网络权限，用于发现并连接局域网内的 NAS 设备，包括 SMB 服务和设备信息服务。你也可以不使用自动发现，手动输入服务器地址。

3. 在线海报、元数据、字幕和歌词

当你启用在线海报、影片元数据、字幕搜索或歌词搜索时，App 可能会把解析后的媒体标题、年份、季/集信息或搜索关键词发送给你选择的在线服务或 App 内置的刮削中转服务。可能使用的服务包括 TMDB、ASSRT、OpenSubtitles、SubDL、Douban、iTunes、LRCLIB、网易歌词等。不同服务会按照各自的服务条款和隐私政策处理请求。

你可以在 App 设置中关闭或更换这些在线服务。私密空间内的条目仅保存在本机；如果你不主动对私密条目执行在线刮削或字幕搜索，App 不会把这些条目的名称用于在线查询。

4. 账号、密码与 API Key

你输入的 NAS、WebDAV、媒体服务器账号密码以及第三方服务 API Key 会保存在设备的 Keychain 或本地存储中，用于后续连接和调用。App 不会把这些凭据出售或用于广告追踪。

5. iCloud 同步

如果你开启 iCloud 同步，App 可能会通过你的 iCloud 账号同步资料库来源配置、私密列表标记、播放进度和部分偏好设置。账号密码和 API Key 不会通过 App 的 iCloud 同步项上传。

6. 缓存与购买

App 可能会在本机保存海报、字幕、歌词、播放进度和媒体索引缓存，以提升浏览和播放体验。你可以在系统或 App 内清理相关数据。购买和订阅状态由 Apple StoreKit 处理，App 只读取必要的购买结果用于解锁功能。

7. 联系我们

如果你对隐私政策或数据处理有疑问，请通过技术支持页面提供的联系方式与我们联系。
